Tools

 

  • Interlinear Text Editor (ITE), is a text editor for XML documents. ITE allows the glossing of a document showing sentences in an interlinear presentation. It maintains a permanent relation between a lexicon and the gloss used in the corpus. ITE
  • SoundIndex is an authoring tool for synchonizing text and sound: SoundIndex
  • mediacut is a set of 2 cgi scripts (one written in C the other written in PERL) which allows the "cutting" of an audio file (for now, only .mp3 or .wav files). It sends the fragment to the client as a stream with the right mime type mediacut
  • mediaplayer is a Java tool (applet and API) which allows a browser to handle a media file (audio or video). The sources of this tool are on a sourceforge site called fieldling mediaplayer
