Formulaire de métadonnées :

 

Chaque ressource archivée est obligatoirement accompagnée d'un fichier qui la décrit (par exemple, pour un fichier audio : lieu et date de l'enregistrement, nom des locuteurs...). Cela permet un catalogage de la ressource dans l'archive.

Les standards utilisés pour les métadonnées sont le XML, Dublin Core et OLAC .

Un formulaire a donc été développé pour permettre de renseigner les informations nécessaires à l'archivage d'une ressource sans avoir à maitriser directement ces standards.

La validation de ce formulaire crée automatiquement un fichier au format XML à enregistrer sur votre ordinateur (ce fichier portera le nom de la ressource qu'il décrit).

 

Accès au formulaire complet / Accès au formulaire simplifié

Conseil : lors de la préparation de vos documents, et pour en conserver par la suite une copie, placez votre fichier de métadonnées dans le même dossier que la ressource qu'il renseigne.

 

Quelques outils pour aider à la transcription / traduction / synchronisation :


(Attention, ces outils ne sont plus maintenus)

 

  • Interlinear Text Editor (ITE) est un logiciel qui permet aux utilisateurs linguistes d'entrer leurs transcriptions et leurs gloses interlinéaires, en particulier pour annoter des enregistrements sonores. Il permet à l'utilisateur d'entrer et d'éditer l'annotation sur quatre niveaux d'analyse: le texte, la phrase, le mot et le morphème. Les éléments des deux plus bas niveaux (i.e. mot et morphème) sont présentés phrase par phrase dans un format interlinéaire, avec le champs glose aligné sous le champs transcription. Pendant que le texte est glosé, le programme construit un lexique, ajoutant les nouveaux éléments trancrits avec leur glose ou ajoutant les nouvelles gloses pour les éléments transcrits existants. ITE facilite la saisie de l'annotation en proposant des gloses pour les éléments précédement recontrés dans le texte ou dans d'autres textes précédement saisis. Des fonctions pour faire des concordances et des recherche de mots ou de morphèmes sont fournis, facilitant la recherche et l'analyse des textes. Ce logiciel est disponible librement sur le site web : ITE
  • SoundIndex est un programme qui permet de synchoniser une transcription avec un enregistrement: SoundIndex
  • mediacut est un ensemble de 2 scripts cgi (l'un écrit en C l'autre en PERL) qui permettent de "couper" un fichier son (pour le moment des fichiers .mp3 ou des fichiers .wav) et d'envoyer le segment "coupé" comme un flux avec le type mime adéquat sur le client mediacut
  • mediaplayer est un outil Java (applet et API) qui permet à un navigateur d'accéder à un fichier media (fichier son ou video). Les sources de cet outil sont sur le site sourceforge de fieldling mediaplayer
